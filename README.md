# Fazendão Agropecuária

Site institucional de página única do **Fazendão Agropecuária** — Juiz de Fora, MG.
Rações, produtos veterinários, pet shop e itens para o campo, com vendas feitas pelo WhatsApp.

## Sobre

Site estático (HTML + CSS + JS em um único arquivo, sem dependências de build), moderno e responsivo, com foco em direcionar o cliente para o WhatsApp.

## Como usar

Abra o arquivo [`index.html`](index.html) no navegador. Não é necessário instalar nada.

### Alterar o número do WhatsApp

No topo do `index.html`, edite a linha:

```js
const WHATSAPP_NUMERO = "553233210919"; // 55 (país) + 32 (DDD) + número, só dígitos
```

## Estrutura

- `index.html` — o site completo
- `Assets/` — imagens (foto da fachada da loja)
